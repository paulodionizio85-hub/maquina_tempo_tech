# ⏳ Máquina do Tempo Tecnológica

Este projeto é uma aplicação web interativa desenvolvida com **Python** e **Streamlit**, que permite explorar marcos históricos da computação, dos videogames e da telefonia móvel. 

O projeto foi desenvolvido como parte dos meus estudos no curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

## 🚀 Funcionalidades

*   **Navegação por Rotas:** Escolha entre Evolução da Computação, Batalha dos Bits (Games) ou Computação de Bolso.
*   **Linha do Tempo Interativa:** Seleção de anos específicos com exibição de registros históricos e imagens.
*   **Fatos Curiosos:** Gerador aleatório de curiosidades sobre o mundo da tecnologia.
*   **Interface Responsiva:** Layout adaptável para diferentes tamanhos de tela.
*   **Salto Aleatório:** Função de descoberta surpresa de eventos históricos.

## 🛠️ Tecnologias Utilizadas

*   [Python](https://www.python.org/) - Linguagem base do projeto.
*   [Streamlit](https://streamlit.io/) - Framework para criação da interface web.
*   [Git/GitHub](https://github.com/) - Controle de versão e hospedagem do código.
